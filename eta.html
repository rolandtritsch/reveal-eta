Some demos ... - AOC

  • Explain/discuss the problem
  • Review the code
  • Compile/run haskell/eta with stack/etlas
  • stack build :haskell-aoc-day01p1
  • stack run haskell-aoc-day01p1
  • etlas build haskell-aoc-day01p1
  • etlas run haskell-aoc-day01p1
  • This works, because stack and etlas (under the hood) both use cabal
  • Main difference is ... performance :)
  • eta is (much) slower than (native) haskell
  • (On average/Not too surprisingly) Its is probably also slower than Scala (in some/a lot of cases).
  • It is hard to say what the real difference is (e.g. warming up the JVM) and how much the difference actually matters to you.
  • In larger test I have seen a difference of (just) factor 2 (1200 vs 2200 secs).
